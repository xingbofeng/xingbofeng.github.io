## About / 关于我

Hi, I'm **Counterxing**.

你好，我是 **Counterxing**。

I started frontend development in **2015**, graduated from the **University of Electronic Science and Technology of China** in **2018**, later worked at **Tencent** and **DJI**, and am now an independent developer.

我从 **2015** 年开始接触前端，**2018** 年毕业于 **电子科技大学**，后续先后在 **腾讯** 和 **大疆** 工作，现在是自由开发者。

For years, I have built complex frontend systems, engineering tools, and product-level web applications, with deep experience in medical imaging, image/video processing, infinite canvas image engineering, and frontend visualization systems. Since **2025**, I have shifted a large part of my energy to **AI Agent development**, exploring agentic coding workflows, tool orchestration, context engineering, and the new collaboration model between humans and software.

过去几年，我持续建设复杂前端系统、工程工具和产品级 Web 应用，并在医学影像、图像/视频处理、无限画布图像工程、前端可视化系统等方向积累了较深经验。自 **2025** 年起，我将更多精力投入到 **AI Agent 开发**，关注 Agentic Coding 工作流、工具编排、上下文工程，以及人与软件之间新的协作模式。

I am an **AI Agent enthusiast, frontend visual engineering expert, former Tencent and DJI engineer, technical writer, open-source enthusiast, and independent developer**. My current focus is turning agents from “chatting models” into engineering collaborators that understand context, use tools, and participate in real development workflows.

我是一名 **AI Agent 狂热者 / 前端视觉工程专家 / 前腾讯 / 前大疆 / 技术博主 / 开源爱好者 / 自由开发者**。现在我尤其关注如何把 Agent 从“能聊天的模型”推进到“能理解上下文、调用工具、参与真实工程流程的协作者”。
